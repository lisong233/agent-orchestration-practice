# AI 军团 — Agent 编排作业

> 基于训练数据，构建多 Agent 编排系统：读取业务文档 → 发现隐含判断规则 → 输出审核结果 + 命中规则 + 判断依据

## 核心任务

1. **规则发现**：从 20 个训练样本（通过 4 / 不通过 16，含 2 个损坏以 rescue 文本替代）中自动发现审核规则
2. **Agent 编排**：用 LangGraph 编排多 Agent 管线（文档解析 → 特征提取 → 规则匹配 → 审核决策）
3. **泛化能力**：少样本学习方法处理高度不平衡的标签分布（通过率 ~20%）

## 训练集（2026-06-24 归拢）

```
训练集/

    └── *.xlsx → ../raw/*.xlsx  ← 标签文件
```

> 所有 `.doc` 已转为 `.docx`，不再有老格式。`可用/` = P1 解析入口，零 `.doc`。
